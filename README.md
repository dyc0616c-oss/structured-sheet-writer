# 表格结构化写入工具

把一批数据安全整理进 CSV 表格，先预览变化，确认后再写入。

## 它能解决什么问题

批量整理表格时，最常见的问题是列名混乱、重复数据和误覆盖。这个 Skill 可以帮助你：

- 把零散数据整理成统一表格；
- 根据编号、手机号等唯一信息识别重复记录；
- 区分哪些是新增数据，哪些会更新原记录；
- 在真正修改文件前先展示变化；
- 发现缺少关键字段或格式不一致的数据。

## 你可以这样使用

安装：

```bash
npx skills add https://github.com/dyc0616c-oss/structured-sheet-writer
```

然后直接对 Agent 说：

```text
把这批客户数据整理进 CSV，先告诉我会新增和更新多少条，不要立即写入。
```

```text
以员工编号作为唯一标识，检查新旧两份数据是否有重复。
```

```text
我确认预览结果了，把数据写入目标表格。
```

## 适合谁

- 经常整理名单、客户或运营数据的人；
- 需要合并多批表格的小团队；
- 担心批量操作误覆盖原数据的用户。

## 使用提醒

- 默认只预览，不会直接修改表格；
- 只有得到确认后才会写入；
- 只读取你明确提供的文件；
- 当前版本处理本地 CSV，不直接连接在线表格账号。
